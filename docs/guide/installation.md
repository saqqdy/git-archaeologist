# Installation

## Option 1: Claude Code Skill

```bash
cd your-project
git clone https://github.com/saqqdy/git-archaeologist.git .git-archaeologist
cp -r .git-archaeologist/.claude/skills/ .claude/skills/
```

## Option 2: NPM Package

```bash
pnpm add git-archaeologist
```

## Option 3: Clone and Run Examples

```bash
git clone https://github.com/saqqdy/git-archaeologist.git
cd git-archaeologist
pnpm install
npx tsx examples/basic-usage.ts
```

# Installation

## Option 1: Claude Code Skill

```bash
cd your-project
git clone https://github.com/saqqdy/git-unearth.git .git-unearth
cp -r .git-unearth/.claude/skills/ .claude/skills/
```

## Option 2: NPM Package

```bash
pnpm add git-unearth
```

## Option 3: Clone and Run Examples

```bash
git clone https://github.com/saqqdy/git-unearth.git
cd git-unearth
pnpm install
npx tsx examples/basic-usage.ts
```
